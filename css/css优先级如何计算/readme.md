- 优先级就近原则， 同权重情况下样式定义最近者为准；
- 载入样式以最后载入的定位为准
优先级为：
    同权重： 内联样式表(标签内部) > 嵌入样式表 > 外部样式表
    !important > id > class > tag
    important 比 内联优先级高
- css选择器分类：
        基本的：
            1. id选择器(id="name")
            2. 类选择器(class="head")
            3. 标签选择器(body, div, ul, li)
            4. 全局选择器(*)
