# 为什么ssr更利于seo
- 因为spa是单页应用， 当请求页面时， 返回的body里为空， 之后执行js将html结构注入到body里， 结合css显示出来
- 但是使用了ssr之后， 请求页面时， 返回的body里已经有了首屏的html解构， 之后结合css显示出来， 而服务端渲染做的是也就是服务器端先渲染生成html， 再发送到客户端， 最后调和为客户端上完全交互的应用程序
